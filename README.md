# RobotTour
Robot Tour codes for the Science Olympiad. If you are doing science olympiad yourself, I ask that you do not use this code to compete, only to compare to one's own work for whatever reason, or get ideas from it.

This code assumes that you are using mechanum wheels on the Rev Robotics hex motors and will work accordingly

#### Setup
1. Hook up the breadboard to ground and power as normal
2. Attach a button to the breadboard with the proper wiring
3. Wire the motors up to the H-bridges as normal
4. Wire encoder pins to ground, power, and two of the input pins on the arduino
5. Configure the pin setup in the code
6. Give the path you want it to follow in the path[] variable as shown
7. Upload and run :D
